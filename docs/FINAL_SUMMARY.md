## 🚀 如何使用

### 安装依赖
```bash
npm install
```

### 开发命令
```bash
npm run dev          # 启动开发服务器（http://localhost:3000）
npm run build        # 构建生产版本
npm run preview      # 预览生产版本
npm run lint         # 检查代码
npm run lint:fix     # 自动修复代码问题
npm run format       # 格式化代码
```

### 音效系统
- 点击顶部导航栏右侧的 🔊 按钮开启/关闭音效
- 音效会在答题、提示、转动等操作时自动播放
